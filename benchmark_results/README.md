[`bbh.csv`](bbh.csv), [`gpqa.csv`](gpqa.csv), and [`mmlu.csv`](mmlu.csv) each contain raw benchmark results: `score` indicates the result of LLM `llm_id` on item `item_id` in trial `trial_id`. See the paper for more details on the benchmark tests, evaluated models, and benchmark evaluation protocols. [`gpqa_info.csv`](gpqa_info.csv) contains additional information on GPQA Diamond items used in the analysis.
